Implement a generateAndPrintResume() engine that fetches JSON data and maps it to a strict, ATS-friendly A4 layout.
Required Sections & Logic:
Header: Centered Name (3xl bold), Role, and inline icons for Location, Email, GitHub, LinkedIn, and Kaggle.
Summary: 2-3 lines of concise professional summary.
Education: List degree, institution, location, and years.

/index.html - Minimal entry point with injection containers.
/assets/js/main.js - Logic for navigation, modal management, and the Resume engine.
/assets/css/styles.css - Custom theme variables and print overrides.
/data/ - Modular JSON files for every section of the site.
