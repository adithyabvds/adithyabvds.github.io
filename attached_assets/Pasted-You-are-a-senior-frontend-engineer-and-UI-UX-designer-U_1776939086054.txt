You are a senior frontend engineer and UI/UX designer.

Upgrade the PROJECTS section and CERTIFICATIONS section to a premium, modern, product-level UI.

---

🎯 GOAL

* Make projects visually impressive and interactive
* Improve hierarchy and readability
* Add certificate links and verification UI
* Maintain JSON-driven system

---

🔥 PART 1: PROJECT SECTION REDESIGN

---

1. CARD STRUCTURE (IMPORTANT)

Each project card must include:

* Title (bold, large)
* Short description (2 lines max)
* Tech stack (badges)
* Primary CTA: "View Details"

Cards must follow:

* Clean layout
* Proper spacing
* Clear hierarchy

Cards are used to present information in modular, scannable units and should not be overloaded with content ([Eleken][1])

---

2. CARD DESIGN

* Rounded corners (16px–20px)
* Subtle shadow
* Hover effect:

  * translateY(-6px)
  * scale(1.02)
* Add soft glow on hover

---

3. CARD INTERACTION

On hover:

* Show additional info (like tags or stats)
* Slight animation

On click:

→ Open modal

---

🔥 PART 2: MODAL REDESIGN

---

Modal must look premium:

* Glass effect (backdrop blur)
* Better spacing
* Structured layout:

LEFT:

* Title
* Role
* Overview

RIGHT:

* Tech stack
* Deliverables

BOTTOM:

* Buttons:

  * Source
  * Live Dashboard

---

🔥 PART 3: CERTIFICATIONS UPGRADE

---

1. JSON UPDATE

Each certificate must include:

{
"title": "",
"issuer": "",
"date": "",
"link": "",
"public": true
}

---

2. UI DESIGN

Each certification card must include:

* Title (bold)
* Issuer (smaller text)
* Date (right aligned)
* "View Certificate" button

---

3. BUTTON

* Label: "View Certificate"
* Opens link in new tab
* Styled as secondary button

---

4. VISUAL IMPROVEMENT

* Add badge/icon (verified feel)
* Add hover effect
* Add spacing between items

Certifications are important signals of professional credibility and should be clearly visible in portfolios ([Kaarwan][2])

---

🔥 PART 4: UX IMPROVEMENTS

---

* Improve spacing between sections
* Make headings more bold
* Add subtle divider lines
* Improve alignment

---

⚠️ RULES

* DO NOT break JSON structure
* Keep everything dynamic
* DO NOT hardcode values

---

🎯 FINAL RESULT

Projects section should feel like:

* SaaS dashboard
* Interactive and modern

Certifications section should feel like:

* Verified credentials list
* Clean and clickable

---

Now implement the upgrade.
